## Նկարագրություն

**Foodcourt-ի համար e-commerce վեբ կայք** — արևելյան (հայկական) խոհանոցի արտադրանքի առցանց վաճառք (պիդե, ուտեսներ), առաքումով (նախնական՝ Երեւան)։ Նախագիծը լուծում է ապրանքների ցուցադրում, պատվերների ձևակերպում, հաճախորդի հաշվի և ադմինի կառավարման խնդիրը մեկ fullstack հավելվածով։

## Stack (որոշված)

- **Fullstack Next.js** (App Router + API Routes), Vercel-ում
- **Frontend.** Next.js 15, React 19, TypeScript (strict), Tailwind CSS
- **Backend.** Next.js Route Handlers, Prisma ORM
- **ԲԴ.** PostgreSQL (Neon / Vercel Postgres)
- **Auth.** NextAuth (Credentials, JWT)

Մանրամասն — [01-ARCHITECTURE.md](./01-ARCHITECTURE.md), [PROJECT_CONTEXT.md](./PROJECT_CONTEXT.md).
