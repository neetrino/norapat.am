## Ընդհանուր նկարագրություն (Overview)

Նպատակն է ստեղծել Foodcourt-ի համար e-commerce վեբ կայք, որն ունի՝

- Լիարժեք **public shop**
- **Customer account** system
- **Հզոր admin panel**
- **Պարզ և արագ checkout**
- **Analytics** և վերահսկում

## 7️⃣ Ադմին պանել (Management System)

### 7.1 Ապրանքի կառավարում

- Ավելացնել / Խմբագրել / Ջնջել ապրանքները

### 7.2 Պատվերի կառավարում

- Պատվերների ցանկ
- Ֆիլտրել ըստ՝ Նոր, Ընթացքում, Առաքված, Չեղարկված
- Պատվերի մանրամասների դիտում
- Պատվերի ստատուսի խմբագրում

### 7.3 Զեղչեր

- Promo code-ի կառավարում
- Զեղչերի տեղադրում և կառավարում

### 7.4 Կատեգորիաներ

- Կատեգորիաների ավելացում / Խմբագրում / Ջնջում

### 7.5 Անալիտիկա

**Ընդհանուր**

- Total orders
- Total revenue
- Average order value

**Պատվերների անալիտիկա**

- Orders by status
- Today / This week / This month

**Ապրանքների անալիտիկա**

- Top 5 best-selling products
- Least-selling products

**Հաճախորդների անալիտիկա**

- New customers
- Repeat orders
- Top customers by spend
